|-----------|------|-------------|

| 🏆 **성장의 쾌감** | 점점 강해지는 체감 | 콤보, 카드 시너지, 메타 업그레이드(영구 해금) |
| 🔄 **리플레이 가치** | 같은 게임을 두 번 플레이핏 않음 | 시드 런, 데일리 챌린지, 9가지 엔딩, 히든 보스 |

# 제1부: INFERNO — 지옥편 ✅

## 1. 디자인 철학 (Design Philosophy)

Abyssos는 단테의 신곡 「지옥편(Inferno)」을 모티브로 한 **전략적 로그라이크 보드게임**입니다.

### 핵심 재미 원칙

| 원칙 | 설명 |
|------|------|
| 🎯 **다양한 액션** | 몬스터(타이밍), 수문장(고유 미니게임), 이벤트(선택) — 같은 패턴 반복 없음 |
| ⚡ **의미 있는 선택** | 안전 vs 도박, 아이템 사용 타이밍, 루트 선택 — 플레이어 판단이 승패 좌우 |
| 📈 **성장의 쾌감** | 수호카드 중첩, 콤보 누적, 스탯 상승 — 점점 강해지는 체감 |
| 💀 **긴장과 위기** | HP 하락 시 위기 모드, 수문장 난이도 상승, 한 방의 역전 |
| 🌟 **서사적 여정** | 9층 지옥을 통과하는 단테의 여정 — 각 층마다 다른 분위기와 도전 |

## 2. 게임 흐름 (Game Loop) — NEW V0.2

```
[시작] → [👹 악마 도전 미니게임] → [보드 이동]
    ↑                              ↓
    |                      ┌─ 👾 몬스터: 15종 미니게임
    |                      ├─ 🤔 이벤트: 서사 분기 이벤트
    |                      ├─ 💎 복병: 강화 몬스터
    |                      ├─ 🎁 별빛 상자: 랜덤 보상
    |                      └─ ⚔️ 상인: 카드/소모품 거래
    |                             |
    |                      [12칸 도달 → 수문장 보스전]
    |                             |
    |                      👹 수문장 보스전 (9종 고유)
    |                      3단계 변신 + 고유 패턴
    |                             |
    |                      승리 → ✨ 수호카드 획득 + 영혼석 → 다음 층
    |                      패배 → 💀 후퇴 + 저주카드 획득 → 재도전
    |                             |
    |                      1층 수문장 처치 → 🏆 지옥 탈출!
```

### 2.1 이동 시스템 — 악마 도전 (Demon Duel)

주사위 없이 **5종 회전 미니게임**으로 이동을 결정합니다. 매 턴 다른 게임이 등장합니다.

| 미니게임 | 장르 | 점수 기준 | 보상 |
|----------|------|----------|------|
| 👹 Demon Whack | 두더지 잡기 | 잡은 악마 수 | 점수 = 이동 칸 수 |
| 🎯 Soul Timing | 타이밍 스톱 | 초록존 정확도 | 정확도 높을수록 별도 별 보상 |
| ⚖️ Snap Judgement | 순간 판단 | 올바른 판단 수 | 연속 성공 시 콤보 복리 |
| ✨ Ember Path | 경로 기억 | 정확히 연결한 노드 수 | 완벽 클리어 시 별빛 상자 |
| 🃏 Sinner Pairs | 짝 맞추기 | 시도 횟수 | 최소 시도 시 추가 보상 |

**이동 거리 = 미니게임 점수 (1~6칸, 실력 비례)**

**완벽 클리어 별 시스템:**
- ⭐ Perfect: 이동력 +1 + 별빛 상자 획득
- ⭐⭐ Flawless: 이동력 +2 + 별빛 상자 + 소모품
- ⭐⭐⭐ Divine: 이동력 +3 + 별빛 상자 + 소모품 + 추가 턴

### 4.4 카드 시너지

- **얼음 심장 + 분노의 투구**: 피해 -3 + D6+1 = 탱킹 시너지
- **피의 축복 + 케류베로스 목줄**: 위기 상황에서 극대화
- **황금 날개 + 폭풍의 은총**: 이동력 극대화 (최대 +4)
- **베르길리우스 지팡이**: 모든 수호카드 효과 50% 증폭
- **NEW: 피의 계약(저주) + 피의 축복(수호)**: 전투 승리 시 HP 3배 회복 but 전투 시작 -3
- **NEW: 거짓된 가면(저주) + 가면술사 비밀(수호)**: 70% 자동 승리, 30% 자동 패배
