## 1. 게임 개요

**Abyssos**는 9층 지옥을 탈출하는 모바일 액션 로그라이크 게임입니다.

- 🎲 **주사위**로 이동
- 🎯 **타이밍 슬라이더**로 몬스터 처치
- ⚡ **연속 처치**로 콤보 쌓기
- 🤔 **선택의 기로**에서 운명을 결정
- 🃏 **카드 찾기**로 수문장 돌파
- ✨ **수호카드**를 모아 강해지기

### 핵심 재미 요소

| 요소 | 설명 |
|------|------|
| 🔥 **콤보 시스템** | 연속 처치 시 보상 증가, 5콤보 달성 시 특별 효과 |
| ⚡ **선택의 순간** | 안전 vs 도박 — 플레이어의 판단이 승패를 가른다 |
| 💀 **위기의 순간** | HP 25% 이하에서 공격력 상승, 빨간 화면 긴장감 |
| 🌟 **크리티컬** | 타이밍 완벽 적중 시 화면 진동 + 2배 보상 |
| 👑 **엘리트 몬스터** | 랜덤 등장, 강하지만 3배 보상 |
| ✨ **수호카드** | 9종 수집, 영구 버프 + 시너지 |

## 4. 액션 시스템

### 4.1 타이밍 슬라이더 (몬스터 전투)

```
[🔴 MISS | 🟡 HIT | 🟢 CRIT | 🟡 HIT | 🔴 MISS]
                  ▲ 인디케이터 왕복
```

- **CRIT**: 인디케이터가 초록존에 정지 → D6=6, 2배 보상, 화면 진동
- **HIT**: 노랑존 → D6=5, 일반 승리
- **MISS**: 빨강존 → D6=2, 패배

### 4.2 카드 찾기 (수문장)

- 3~5장 카드 중 정답 카드 찾기
- 정답 → 승리 + 수호카드 획득
- 오답 → 패배 + 후퇴

### 4.3 선택의 기로

```
🤔 갈림길에 섰다!

[🛡 안전한 길]        [⚡ 위험한 지름길]
HP +10 확정           HP -10 or HP +25
```

## 5. 콤보 시스템

| 콤보 | 효과 |
|------|------|
| 3연속 처치 | 보상 +20%, "🔥 ON FIRE!" |
| 5연속 처치 | 보상 +50%, 초록존 +5%, "⚡ UNSTOPPABLE!" |
| 7연속 처치 | 모든 몬스터 전투력 -1, "👑 GODLIKE!" |
| 피격/패배 | 콤보 리셋 |

## 6. 위기 시스템 (HP ≤ 25%)

- 화면 테두리 붉게 점멸
- 공격력 +1 (D6 보정)
- "💀 치명상! 힘이 솟는다!"

## 7. 엘리트 몬스터

- 각 층당 1마리 (7번 타일 고정)
- 일반 몬스터보다 전투력 +2
- 보상 3배, 콤보 +2 카운트
