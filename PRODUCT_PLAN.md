# 제1부: INFERNO — 지옥편 ✅

## 1. 디자인 철학 (Design Philosophy)

Abyssos는 단테의 신곡 「지옥편(Inferno)」을 모티브로 한 **전략적 로그라이크 보드게임**입니다.

### 핵심 재미 원칙

| 원칙 | 설명 |
|------|------|
| 🎯 **다양한 액션** | 몬스터(타이밍), 수문장(고유 미니게임), 이벤트(선택) — 같은 패턴 반복 없음 |
| ⚡ **의미 있는 선택** | 안전 vs 도박, 아이템 사용 타이밍, 루트 선택 — 플레이어 판단이 승패 좌우 |
| 📈 **성장의 쾌감** | 수호카드 중첩, 콤보 누적, 스탯 상승 — 점점 강해지는 체감 |
| 💀 **긴장과 위기** | HP 하락 시 위기 모드, 수문장 난이도 상승, 한 방의 역전 |
| 🌟 **서사적 여정** | 9층 지옥을 통과하는 단테의 여정 — 각 층마다 다른 분위기와 도전 |

```

                                              ├─ 🤔 선택의 기로: 도박 선택
                                              ├─ ✧ 휴식: HP 회복
                                              ├─ ⚠ 함정: 회피 액션 또는 피해
                                              ├─ 💎 보물: 보너스 획득
                                              └─ 👑 엘리트: 강화 몬스터
                                                     │
                                           [12칸 도달]
                                                │
                                           👹 수문장 보스전

                                                     │
                                           1층 수문장 처치 → 🏆 탈출! → 연옥편 계속
```

## 3. 액션 시스템 (Action Systems)

### 3.0 주사위 듀얼 — 악마 VS 사용자 (Demon Dice Duel)

모든 주사위는 이제 **악마와 사용자의 대결**로 진행됩니다. 단순히 주사위를 굴려 이동하는 것이 아니라, 악마의 주사위를 이겨야만 이동할 수 있습니다.

```
┌───────────────────┐     ┌───────────────┐     ┌───────────────────┐
│   👤 사용자 주사위  │  VS  │   👹 악마 주사위   │  →  │  승자가 이동       │
│   2D6 굴림         │     │   2D6 + 보너스    │     │  패자는 피해 + 대기  │
└───────────────────┘     └───────────────┘     └───────────────────┘
```

**승리 규칙:**
- 더블이 나오면 더블이 아닌 주사위를 무조건 이깁니다
- 둘 다 더블이면 더 높은 더블이 승리
- 둘 다 더블이 아니면 합계가 높은 쪽이 승리
- 무승부는 악마의 홈 어드밴티지로 악마 승리

**악마 보너스 (층별):**
- 9층~8층 (가장 깊은 지옥): 악마 +2~3 보너스
- 7층~5층 (중간 지옥): 악마 +1 보너스
- 4층~3층 (상층 지옥): 보너스 없음
- 2층~1층 (지상 근접): 사용자 우위 (악마 -1)

**결과:**
- **사용자 승리**: 주사위 합계만큼 이동
- **사용자 더블 승리**: 주사위 합계만큼 이동 + 더블 카운트 증가
- **악마 승리**: 이동 불가 + 3 피해
- **악마 더블 승리**: 이동 불가 + 5 피해

이 시스템은 연옥편과 천국편에도 동일하게 적용되며, 각 편의 테마에 맞게 악마 대신 수호천사/대천사가 대결 상대로 변경됩니다.

### 3.1 미니게임 — 몬스터 전투 (15종)

TimingSlider를 완전히 제거하고, 각 원소별로 **완전히 다른 장르**의 미니게임을 적용했습니다.

| 원소 | 미니게임 | 장르 | Tier A (하급) | Tier B (상급) |
|------|---------|------|---------------|---------------|
| ❄️ 얼음 | **NumberMemory** | 숫자 암기 | 3자리, 2.5초 | 5자리, 1.8초 |
| 🎭 환영 | **CardMatch** | 카드 기억 | 3장 찾기 | 4장 찾기 |
| 🩸 피 | **WhackMole** | 두더지 잡기 | 6마리/10초 | 10마리/8초 |
| 🔥 불 | **MathPuzzle** | 산수 문제 | 중급 (8초) | 상급 (6초) |
| 💢 분노 | **SpeedTyping** | 단어 타이핑 | 중간 단어 | 긴 단어 |
| 💰 탐욕 | **DiceBet** | 주사위 베팅 | 3판 2승 | 4판 3승 |
| ☠ 독 | **ReflexCatch** | 낙하 캐치 | 넓은 존, 느림 | 좁은 존, 빠름 |
| 🌪 바람 | **DirectionDodge** | 방향 회피 | 7회 중 4회 | 10회 중 6회 |
| ✨ 신성 | **BalanceScale** | 균형 유지 | 4초, 낮은 감도 | 5초, 높은 감도 |

**난이도 체계:**
- **Tier A (하급)**: 더 긴 제한시간, 더 적은 목표치, 더 넓은 성공 범위
- **Tier B (상급)**: 더 짧은 제한시간, 더 많은 목표치, 더 좁은 성공 범위
- **엘리트**: Tier B 기반 + 추가 패널티 (power +2, reward ×3)

**미니게임 성공 = 전투 승리 보장** — 미니게임 통과 시 몬스터 power 이상의 D6 값이 자동 설정됩니다.

### 3.4 이벤트 시스템

| 타일 | 이벤트 | 내용 |
|------|--------|------|
| 🤔 | **선택의 기로** | 플레이어가 직접 선택: 🛡 안전(소량 회복 확정) vs ⚡ 도박(50% 대량 회복/50% 피해) |
| ✧ | **휴식** | HP +10~20 자동 회복 |

### 3.5 주사위 승리 → 자동 이동 (Auto-Move)

주사위 듀얼에서 승리 시 **즉시 자동으로 이동**합니다. 별도의 Move 버튼이 필요 없으며, 주사위 결과에 따라 바로 다음 페이즈(몬스터 전투/이벤트/문지기/다음 턴)로 전환됩니다.

```
🎲 Roll → 승리 시 즉시 이동 → 몬스터/이벤트/문지기/다음턴
         → 패배 시 피해 + 턴 소비 → 다시 Roll
```

## 4. 수호카드 시스템 (Guardian Cards)

### 4.1 디자인 원칙

- 수문장 처치 시 해당 층의 **수호카드** 획득
- 각 카드는 **패시브 효과** + **액티브 능력** 보유
- 액티브 능력은 **턴당 1회** 사용 가능 (일부는 쿨다운)
- 수호카드는 **중첩**되어 시너지 발생

### 4.2 9종 수호카드

| 층 | 수호카드 | 패시브 효과 | 액티브 능력 | 쿨다운 |
|----|----------|------------|------------|--------|
| 9 | ❄️ 얼음 거인의 심장 | 받는 피해 -2 | **빙벽**: 이번 전투 피해 완전 차단 | 5턴 |
| 8 | 🎭 가면술사의 비밀 | 20% 확률 적 능력 무효 | **전투 스킵**: 몬스터 전투 1회 회피 | 1회용 |
| 7 | 🩸 피의 축복 | HP 30%↓ → D6+1 | **피의 갈증**: 승리 시 HP 10 추가 회복 | 3턴 |
| 6 | 🔥 심문관의 등불 | 적 능력 사전 표시 | **진실의 빛**: 퀴즈 정답 하나 공개 | 2턴 |
| 5 | 💢 분노의 투구 | 전투 D6 +1 | **광폭화**: 이번 전투 D6 +2 (HP 회복 -50%) | 4턴 |
| 4 | 💰 황금의 날개 | 이동력 +1 | **비행**: 3칸 전진 | 3턴 |
| 3 | ☠ 케르베로스의 목줄 | D6 2회 굴려 선택 | **파수견 소환**: 전투 자동 승리 | 5턴 |
| 2 | 🌪 폭풍의 은총 | 최소 주사위 3 | **순풍**: 주사위 +3 | 2턴 |
| 1 | ✨ 베르길리우스의 지팡이 | 모든 효과 +50% | **지혜의 인도**: 모든 액티브 쿨다운 리셋 | 1회용 |

### 4.3 수호카드 시너지

- **얼음 심장 + 분노의 투구**: 피해 -3 + D6+1 = 탱킹 시너지
- **피의 축복 + 케르베로스 목줄**: 위기 상황에서 극대화
- **황금 날개 + 폭풍의 은총**: 이동력 극대화 (최대 +4)
- **베르길리우스 지팡이**: 모든 수호카드 효과 50% 증폭

## 9. 연옥 세계관: 정화의 산

연옥산(Mount Purgatory)은 남반구 바다 한가운데 솟은 거대한 산입니다. 단테의 「연옥편(Purgatorio)」에서 영혼들은 이 산의 7개 테라스에서 죄를 정화한 후, 정상의 지상낙원(Earthly Paradise)에서 베아트리체를 만납니다.

> **지옥과의 핵심 차이**: 지옥은 죄의 **형벌( punishment)**, 연옥은 죄의 **정화(purification)** 입니다. 같은 주제라도 고통이 아닌 치유와 성찰의 과정입니다.

## 12. 연옥 보드 디자인

**산 오르기 메커니즘**:
- 각 테라스는 **8칸** (지옥보다 짧지만 밀도 높음)
- 플레이어는 **전진만 가능** — 후퇴 없음 (정화의 방향은 오직 위로)
- 각 테라스 말미에 **정화의 문(Purification Gate)** — 수호천사와의 만남
- 전체 7층 × 8칸 = **56칸** + 7개의 정화의 문

```
테라스 타일 구조: A - B - 선택 - A - B - 기도 - 정화 - 선택 → 😇 수호천사
```

## 15. 정화카드 (Purification Cards, 7종)

천사 처치 시 획득하는 카드. 지옥의 '수호카드'보다 강력하지만 더 조건적인 효과를 가집니다. 지옥의 수호카드(9장) + 연옥의 정화카드(7장) = 최대 16장의 카드가 중첩됩니다.

| 층 | 정화카드 | 주 효과 | 부 효과 |
|----|----------|---------|---------|
| 1층 | 🪨 겸손의 돌 (Stone of Humility) | 전투 패배해도 후퇴하지 않음 | 받는 크리티컬 피해 -50% |
| 2층 | 👁 자비의 눈 (Eye of Mercy) | 적 특수능력 50% 확률 무효 | 함정 완전 면역 |
| 3층 | 🌬 온유의 숨결 (Breath of Gentleness) | 전투 전 적 전투력 -1 | 독/저주 지속 시간 -1턴 |
| 4층 | ⚡ 열정의 발걸음 (Step of Zeal) | 더블 시 3연속 이동 가능 | 스턴 완전 면역 |
| 5층 | 🤲 청빈의 손 (Hand of Poverty) | HP 50% 이하 시 D6 +2 | 전투 보상 HP +50% |
| 6층 | 🍎 절제의 입맞춤 (Kiss of Temperance) | 회복 아이템 효과 2배 | 전투 D6 2회 굴리기 |
| 7층 | 🔥 순결의 불꽃 (Flame of Purity) | 모든 정화카드 효과 2배 | 획득 즉시 지상낙원 개방 |

### 정화카드 + 수호카드 시너지 예시

- **겸손의 돌 + 얼음 심장**: 패배해도 후퇴 없음 + 피해 감소 → 무적 탱킹
- **자비의 눈 + 가면술사의 비밀**: 70% 확률로 적 능력 무효 → 전투 무력화
- **열정의 발걸음 + 황금 날개**: 더블 시 3연속 + 비행 → 최대 6칸 이동
- **순결의 불꽃 + 베르길리우스 지팡이**: 모든 효과 3배 증폭 → 신급 시너지

## 19. 천국편 게임 디자인 철학

### 지옥·연옥과의 완전한 차별화

| 요소 | 지옥 (Inferno) | 연옥 (Purgatorio) | 천국 (Paradiso) |
|------|---------------|-------------------|-----------------|
| **이동 방향** | 제한적 전진 | 전진만 | **자유 비행** (원하는 칸으로) |
| **전투** | 강제 전투 | 선택적 전투 | **축복 중심** (전투 최소화) |
| **위험** | 함정, 패배, 후퇴 | 죄의 무게 | **없음** — 실패해도 손실 없음 |
| **목표** | 생존하며 탈출 | 정화하며 등반 | **경이를 체험하며 승천** |
| **체력** | HP (100 시작, 감소) | HP (지옥에서 계승) | **은총 (Grace)** — 감소하지 않음 |
| **실패** | HP 0 → 게임 오버 | 전투 패배 → 재도전 | **실패 없음**, 은총만 덜 쌓임 |
| **난이도** | 중간 (생존 중심) | 중상 (선택 중심) | **경험 중심** (난이도보다 서사) |

> **핵심 아이디어**: 천국에서는 '게임 오버'가 없습니다. 대신 플레이어는 **얼마나 충만하게 천국을 경험했는가**로 평가받습니다. 지옥의 공포, 연옥의 노력을 거친 후 천국에서의 해방감과 경이로움이 최종 보상입니다.

## 20. 천국편 신규 시스템: 은총 게이지 (Grace Gauge)

HP를 대체하는 천국편 전용 시스템입니다.

- **최대치**: 100 (시작 50)
- **획득 방법**: 축복, 천사와의 만남, 대천사 시험 성공
- **소모**: **없음** — 천국에서는 은총이 감소하지 않습니다
- **진정한 엔딩 조건**: 은총 100 + 9천 클리어
- **초과 가능 여부**: 🌟 별의 망토(8천 성물) 보유 시 최대 150까지

은총 게이지가 높을수록 더 많은 선택지와 숨겨진 이벤트가 해금됩니다.

## 21. 천국편 이동 시스템: 빛의 날개 (Wings of Light)

- 2D6 주사위 대신 **3D4 빛의 날개** (3~12칸 이동)
- **더블** (같은 눈 2개) → 빛의 축복 (은총 +5)
- **트리플** (같은 눈 3개) → 천사의 개입 (원하는 칸으로 즉시 이동 + 은총 +10)
- 각 천구는 10칸, 말미에 **광휘의 문(Radiant Gate)**

## 33. 게임 스토리 (Complete Game Story)

### Prologue — 코퀴토스의 얼음에서 깨어나다

당신은 지옥의 가장 깊은 곳, **코퀴토스의 얼음 호수** 한가운데서 눈을 뜹니다. 어떻게 여기까지 왔는지 기억나지 않습니다. 단지 살을 에는 추위와 함께, 머릿속에 울리는 알 수 없는 목소리만이 당신을 반깁니다.

> *"길을 잃은 여행자여. 나는 베르길리우스. 그대가 이곳을 빠져나갈 수 있도록 인도하리라. 하지만 먼저 스스로의 힘으로 지옥의 아홉 고리를 통과해야 한다. 각 층의 수문장을 물리치고 그들의 수호카드를 모아라. 그것만이 유일한 탈출의 열쇠다."*

당신은 얼어붙은 손으로 첫 번째 주사위를 집어 듭니다. 악마의 형상이 어둠 속에서 당신의 주사위를 노려봅니다. 지옥의 9개 층을 통과하는 여정이 시작됩니다.

### Act 1 — 지옥편: Inferno (9 Circles)

**9층 — 배반 (Treachery)**: 루치페르의 날갯짓이 얼음을 더욱 단단히 얼립니다. 배신자들의 영혼이 머리부터 발끝까지 얼어붙은 채 영원의 형벌을 받고 있습니다. 얼음 거인이 관문을 지키고 있습니다.

**8층 — 사기 (Fraud)**: 말레볼제의 10개 구렁에서 사기꾼들이 각자 다른 고통을 받습니다. 가면을 쓴 위장의 군주가 진짜 얼굴을 찾아내라고 도전합니다.

**7층 — 폭력 (Violence)**: 끓는 피의 강 플레게톤. 피의 파수꾼이 당신의 속도를 시험합니다. 전투의 상처가 점점 쌓여갑니다.

**6층 — 이단 (Heresy)**: 불타는 석관의 도시. 불꽃 심문관이 신곡의 진리를 묻습니다. 지식만이 이 관문을 통과하는 열쇠입니다.

**5층 — 분노 (Wrath)**: 스틱스의 검은 늪. 분노의 화신이 실패할 때마다 더욱 빨라집니다. 감정을 통제하는 자만이 살아남습니다.

**4층 — 탐욕 (Greed)**: 황금의 무게가 영혼을 짓누릅니다. 황금 골렘이 더 많은 것을 걸라고 속삭입니다. 욕심과 생존 사이에서 선택해야 합니다.

**3층 — 식탐 (Gluttony)**: 진흙비와 악취 속에서 케르베로스가 세 머리로 으르렁댑니다. 세 번의 시험이 당신을 기다립니다.

**2층 — 색욕 (Lust)**: 영원한 폭풍. 폭풍의 정령이 예측 불가능한 바람으로 당신의 균형을 시험합니다. 바람의 의지에 몸을 맡겨야 합니다.

**1층 — 미망 (Limbo)**: 베르길리우스의 그림자가 마지막 시험을 준비합니다. "너는 지옥을 통과할 자격이 있는가?" — 모든 것을 증명해야 합니다.

베르길리우스를 물리치면, 그의 지팡이가 당신의 손에 쥐어집니다. 별이 가득한 하늘이 보입니다. 지옥을 탈출한 것입니다.

### Act 2 — 연옥편: Purgatorio (7 Terraces)

> *"잘 싸웠다. 하지만 여정은 아직 끝나지 않았다."* — 베르길리우스

지옥을 탈출한 당신은 연옥산 기슭에 도착합니다. 하늘에는 별들이 반짝이고, 새벽이 다가옵니다. 이곳의 공기는 다릅니다. 희망의 냄새가 납니다.

**1층 — 교만 (Pride)**: 무거운 돌을 짊어진 영혼들. 겸손의 천사가 돌의 무게를 시험합니다.

**2층 — 시기 (Envy)**: 꿰매진 눈의 길. 자비의 천사가 다른 이의 빛을 시기하지 말라고 속삭입니다.

**3층 — 분노 (Wrath)**: 검은 연기 속에서 분노의 투영이 도전합니다. 온유의 천사가 연기처럼 흘려보내라고 인도합니다.

**4층 — 나태 (Sloth)**: 끝없는 돌계단. 열정의 천사가 번개처럼 달려와 잠든 영혼을 깨웁니다.

**5층 — 탐욕 (Greed)**: 수정 바닥의 길. 청빈의 천사가 빈 손이 가장 큰 부라고 가르칩니다.

**6층 — 식탐 (Gluttony)**: 향기로운 과일나무 아래. 절제의 천사가 진정한 풍요는 절제에서 온다고 말합니다.

**7층 — 색욕 (Lust)**: 정화의 불기둥. 순결의 천사가 불 속을 통과하라고 손짓합니다.

불의 벽을 통과하면, **베아트리체**가 기다리고 있습니다. 베르길리우스는 미소 지으며 사라집니다.

> *"잘 왔다, 나의 사랑하는 영혼이여. 이제 별 너머의 여정이 시작된다."* — 베아트리체

### Act 3 — 천국편: Paradiso (9 Spheres)

베아트리체와 함께 당신은 지상낙원에서 천상으로 솟아오릅니다. 9개의 천구가 빛의 길을 엽니다.

**1천 — 달 천구 (Moon)**: 가브리엘이 충실함을 시험합니다. 빛의 정령들이 은총을 선물합니다.

**2천 — 수성 천구 (Mercury)**: 라파엘이 선택의 지혜를 시험합니다.

**3천 — 금성 천구 (Venus)**: 우리엘이 사랑의 깊이를 시험합니다.

**4천 — 태양 천구 (Sun)**: 미카엘이 용기와 빛을 시험합니다. 찬란한 빛의 향연.

**5천 — 화성 천구 (Mars)**: 사마엘이 희생의 가치를 시험합니다.

**6천 — 목성 천구 (Jupiter)**: 자드키엘이 정의의 균형을 시험합니다.

**7천 — 토성 천구 (Saturn)**: 카시엘이 명상과 침묵을 시험합니다.

**8천 — 항성 천구 (Fixed Stars)**: 메타트론이 지식의 완성을 시험합니다.

**9천 — 원동천 (Primum Mobile)**: 모든 천사들의 합창. 마지막 대천사 산달폰이 최종 승천을 인도합니다.

### Epilogue — 엠피리오 (Empyrean)

모든 천구를 통과한 당신은 **엠피리오** — 모든 빛의 근원에 도달합니다.

> *"이곳이 모든 여정의 끝이자 시작이다. 그대는 지옥을 통과하고, 연옥에서 정화되었으며, 천국의 빛에 도달했다. 신곡의 길을 완주한 영혼이여, 이제 그대 자신이 빛이 되어라."*

당신은 돌아봅니다. 지옥의 어둠, 연옥의 새벽, 천국의 빛 — 모든 것이 하나로 합쳐집니다. 여정은 끝났지만, 진정한 이야기는 이제 시작됩니다.
