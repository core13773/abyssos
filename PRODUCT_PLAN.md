| 재미 원칙 | 설명 | 게임 내 구현 |
|-----------|------|-------------|
| 🎲 **매판 다른 덱** | 로그라이크 덱빌딩 — 40+ 카드 중 랜덤 획득 | 수호카드 + 정화카드 + 성물 + 저주카드 + 소모품 = 조합의 미학 |

| 🔄 **리플레이 가치** | 같은 게임을 두 번 플레이하지 않음 | 3부작 연결, 메타 진행, 9가지 엔딩, NG+ |

# 제1부: INFERNO — 지옥편 ✅

## 1. 디자인 철학

단테의 신곡 「지옥편(Inferno)」을 모티브로 한 **전략적 로그라이크 보드게임**.

| 원칙 | 설명 |
|------|------|
| 🎯 **다양한 액션** | 몬스터(타이밍), 수문장(고유 미니게임), 이벤트(선택) — 같은 패턴 반복 없음 |
| ⚡ **의미 있는 선택** | 안전 vs 도박, 아이템 사용 타이밍, 루트 선택 — 플레이어 판단이 승패 좌우 |
| 📈 **성장의 쾌감** | 수호카드 중첩, 콤보 누적, 스탯 상승 — 점점 강해지는 체감 |
| 💀 **긴장과 위기** | HP 하락 시 위기 모드, 수문장 난이도 상승, 한 방의 역전 |
| 🌟 **서사적 여정** | 9층 지옥을 통과하는 단테의 여정 — 각 층마다 다른 분위기와 도전 |

## 2. 게임 흐름 (Game Loop)

```
[시작] → [👹 악마 도전 미니게임] → [보드 이동]
    ↑                              ↓
    |                      ┌─ 👾 몬스터: 15종 미니게임
    |                      ├─ 🤔 이벤트: 10종 서사/도박/상점 이벤트
    |                      ├─ 💎 복병: 강화 몬스터 (엘리트)
    |                      └─ ⚔️ 상인: 소모품 거래
    |                             |
    |                      [12칸 도달 → 수문장 보스전]
    |                             |
    |                      👹 수문장 보스전 (9종 고유 미니게임)
    |                             |
    |                      승리 → ✨ 수호카드 획득 → 다음 층
    |                      패배 → 💀 후퇴 + HP 피해 → 재도전
    |                             |
    |                      1층 수문장 처치 → 🏆 지옥 탈출!
```

### 2.1 이동 시스템 — 악마 도전 (Demon Duel)

주사위 없이 **10종 회전 미니게임**으로 이동을 결정합니다. 매 턴 다른 게임이 등장합니다.

| 미니게임 | 장르 | 점수 기준 | 이동 거리 |
|----------|------|----------|----------|
| 👹 Demon Whack | 두더지 잡기 | 잡은 악마 수 | 점수 = 이동 칸 수 |
| 🎯 Soul Timing | 타이밍 스톱 | 초록존 정확도 | 정확도에 비례 |
| ⚖️ Snap Judgement | 순간 판단 | 올바른 판단 수 | 성공률에 비례 |
| ✨ Ember Path | 경로 기억 | 정확히 연결한 노드 수 | 연결 수에 비례 |
| 🃏 Sinner Pairs | 짝 맞추기 | 시도 횟수 | 성공 시 이동력 + |
| 🪨 Rock Stack | 탑 쌓기 | 쌓은 높이 | 높이에 비례 (최대 6) |
| ⚡ Lightning React | 번개 반응 | 반응 속도 | 성공 시 고정 이동 |
| 👁 Hidden Word | 숨은 단어 | 찾은 글자 수 | 글자 수에 비례 |
| 🌫 Smoke Maze | 안개 미로 | 출구 도달 여부 | 성공 시 +이동 |
| 💎 Crystal Break | 수정 파괴 | 파괴한 수정 수 | 수에 비례 |

**퍼펙트 클리어 본 시스템:**
- 미니게임 점수 ≥ 5: **✨ 퍼펙트 클리어** → 영혼석 +1
- 연속 전투 승리: **콤보 시스템** → 보상 복리 (3연승 +15%, 5연승 +30%, 7연승 +50%)

# 제2부: PURGATORIO — 연옥편 ✅

## 1. 연옥의 개념

지옥을 탈출한 플레이어는 **연옥(Purgatorio)**에 도착합니다. 여기서는 죄를 정화하며 7개의 테라스(Terrace)를 등반해야 합니다.

| 테라스 | 죄 | 정화 | 테마 |
|--------|-----|------|------|
| 1 | 교만 (Pride) | 짐 지고 등반 | 땅/바위 |
| 2 | 시기 (Envy) | 눈가리개 착용 | 안개/그림자 |
| 3 | 분노 (Wrath) | 연기 속 걷기 | 화산/연기 |
| 4 | 나태 (Sloth) | 지속적인 움직임 | 물/늪 |
| 5 | 탐욕 (Greed) | 황금 포기 | 사막/황금 |
| 6 | 식탐 (Gluttony) | 단식 | 숲/엉겅퀴 |
| 7 | 색욕 (Lust) | 불 속 걷기 | 불/용암 |

## 2. 연옥 게임 루프

```
[시작] → [주사위 굴리기] → [보드 이동]
    ↑                              ↓
    |                      ┌─ 👾 죄의 투영 전투
    |                      ├─ 🤔 연옥 이벤트
    |                      └─ 💎 수호천사 조우
    |                             |
    |                      [7칸 도달 → 수호천사 시험]
    |                             |
    |                      👼 수호천사 전투 (7종 고유)
    |                             |
    |                      승리 → ✨ 정화카드 획득 → 다음 테라스
    |                      패배 → 💀 후퇴 → 재도전
    |                             |
    |                      7층 수호천사 처치 → 🌹 지상낙원!
```

## 3. 연옥 전용 시스템

### 3.1 죄의 투영 (Sin Projection)

각 테라스의 죄를 구현한 몬스터. 지옥의 몬스터보다 강하지만 정화카드 별너스를 받습니다.

### 3.2 수호천사 (Angel Guardian)

| 천사 | 시험 | 보상 |
|------|------|------|
| 1 | 색 순서 기억 | 정화카드-1 + Grace |
| 2 | 패턴 기억 | 정화카드-2 + Grace |
| 3 | 리듬 탭 | 정화카드-3 + Grace |
| 4 | 빠른 탭 | 정화카드-4 + Grace |
| 5 | 균형 유지 | 정화카드-5 + Grace |
| 6 | 색상 추론 | 정화카드-6 + Grace |
| 7 | 최종 시험 | 정화카드-7 + 대량 Grace |

### 3.3 정화카드 (Purification Cards)

연옥 보스 처치 시 획득. 천국편에서 Grace 별너스로 작용.

# 제3부: PARADISO — 천국편 ✅

## 1. 천국의 개념

연옥을 정화한 플레이어는 **천국(Paradiso)**에 도달합니다. 9개의 천구(Sphere)를 통해 빛의 합일을 추구합니다.

| 천구 | 주제 | 대천사 |
|------|------|--------|
| 1 | 달 | 가브리엘 |
| 2 | 수성 | 라파엘 |
| 3 | 금성 | 우리엘 |
| 4 | 태양 | 미카엘 |
| 5 | 화성 | 사마엘 |
| 6 | 목성 | 사달리아 |
| 7 | 토성 | 카시엘 |
| 8 | 별 | 사달폰 |
| 9 | 원动天 | 메타트론 |

## 2. 천국 게임 루프

```
[시작] → [주사위 굴리기] → [보드 이동]
    ↑                              ↓
    |                      ┌─ ✨ 빛의 정령 축복
    |                      ├─ 🤔 천국 이벤트
    |                      └─ 👼 대천사 시험
    |                             |
    |                      [7칸 도달 → 대천사 시험]
    |                             |
    |                      👼 대천사 전투 (9종 고유)
    |                             |
    |                      승리 → ✨ 성물 획득 + Grace
    |                      패배 → 💀 Grace 감소
    |                             |
    |                      9천구 도달 → ☀ 엠피리오!
```

## 3. 천국 전용 시스템

### 3.1 Grace (은총)

천국편의 핵심 스탯. 이벤트와 보스전에서 획득/소모. 성물 획득에 필요.

### 3.2 대천사 (Archangel)

| 대천사 | 시험 | 보상 |
|--------|------|------|
| 가브리엘 | 패턴 기억 | 성물-1 + Grace |
| 라파엘 | 패턴 기억 | 성물-2 + Grace |
| 우리엘 | 리듬 탭 | 성물-3 + Grace |
| 미카엘 | 리듬 탭 | 성물-4 + Grace |
| 사마엘 | 빠른 탭 | 성물-5 + Grace |
| 사달리아 | 색상 추론 | 성물-6 + Grace |
| 카시엘 | 패턴 기억 | 성물-7 + Grace |
| 사달폰 | 리듬 탭 | 성물-8 + Grace |
| 메타트론 | 최종 시험 | 성물-9 + 대량 Grace |

### 3.3 성물 (Celestial Relics)

천국 보스 처치 시 획득. 최종 엔딩에 영향.

| 부활의 축복 | HP 0 시 1회 부활 (HP 30% 회복) | 50석 | 1단계 |

### 8.2 로비 업그레이드 패널 ✅
